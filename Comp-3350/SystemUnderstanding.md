# Understading the System

> What problem does it solve?
This project provides a digital implementation of Sudoku with multiple difficulty levels, allowing users to generate, solve, and save puzzles. It addresses the need for an interactive, persistent game experience with flexibility in storage (filesystem or database).

> What are its major features?
- Interactive Sudoku puzzle solving
- Multiple difficulty levels
- Save and load game functionality
- Sudoku puzzle generation
- User progress tracking
- Multi-language support (English and Polish)
- Integration with both filesystem and database for data persistence

> How does a user interact with it?
Users interact with the application through a graphical user interface (GUI) built using JavaFX. They can select difficulty levels, generate new puzzles, input their solutions, and save their progress. The application also provides options to load previously saved games and switch between languages.
